**Poreia**

This repository contains the source code for my personal portfolio website. It serves as a central hub to showcase my full-stack development projects.

**Technology Stack**

The frontend of this portfolio is built using React version 19 and is bundled with Vite version 8. The user interface is styled utilizing Tailwind CSS version 4

The portfolio highlights my proficiency across a broader technology stack. My primary development languages and frameworks include JavaScript, ReactJS, Python, Django, and Django REST Framework. Database management is handled through PostgreSQL, while data fetching relies on Tanstack Query. For deployment and operational management, I utilize tools such as Git, GitHub, Vercel, Render, Docker, Brevo, Resend, and Cron Jobs.
